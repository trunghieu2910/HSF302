# HSF302 - Tài liệu & Ghi chú môn học

Nơi lưu trữ tài liệu, ghi chú và chia sẻ code cho môn học HSF302.

## 📁 Cấu trúc thư mục

```
HSF302/
├── notes/          # Ghi chú lý thuyết theo từng buổi học
├── code/           # Code mẫu và bài tập thực hành
└── resources/      # Tài liệu tham khảo, slide, đề thi
```

## 📝 Nội dung

- **notes/**: Ghi chú lý thuyết, tóm tắt nội dung từng chương/buổi học
- **code/**: Source code các bài tập, demo và project thực hành
- **resources/**: Slide bài giảng, tài liệu tham khảo và đề thi mẫu
