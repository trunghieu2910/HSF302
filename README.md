## 📂 Cấu trúc Repository
Repository này được sắp xếp để dễ dàng theo dõi tiến độ học tập:

* 📁 **`/Session 1 `**: Chứa source code và hướng dẫn cài maven và xây dựng 1 thư viện maven riêng trên local 
* 📁 **`/Session 2 `**: Ôn tập lại JDBC vaf nhìn ra nhuọc điểm của chúng
* 📁 **`/Session 3`**: Giải pháp thay thế JDBC là ORM /JPA / Hibernate  , ý tưởng codeFirst
* 📁 **`/Session 4`**: SOLID
* 📁 **`/Session 5`**: Spring Framework 

## 🛠️ Công nghệ sử dụng
Vì HSF tập trung vào hệ sinh thái Java Backend, project này chủ yếu sử dụng:
* **Ngôn ngữ:** Java
* **Framework:** Spring Boot, Hibernate
* **Quản lý dự án & Thư viện:** Maven (qua file `pom.xml`)
* **Database:** SQL Server (sử dụng thư viện `mssql-jdbc`)
